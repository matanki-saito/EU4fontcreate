■NvcompressFrontEnd

●作成
sevrunear

●必要ランタイム
.NET Framework 4.5

●これは何？
その名の通り画像ファイルをDDS形式のファイルに変換するnvcompress.exeのGUIフロントエンドです
エフェクト等で大きいサイズの各種テクスチャを使う時に少しでもメモリサイズを稼ぎたくなったら使ってみると良いかもしれません

●注意
適当にテストしただけなので、変なことをすると落ちるかもしれません
入力ファイルが消えたりすることはまずないと思いますが、何か起きても責任は取れませんし取りません
このことに同意する場合のみご使用ください

●入力対応形式
jpeg、png、bmp、tga、psd

●出力対応形式
DXT1、DXT1a、DXT1normalmap、DXT3、DXT5、DXT5normalmap、ATI1、3Dc/ATI2、RGBA


●再配布
する人もいないだろうし大したもんではないですが面倒を避けるため許可しません
配布場所にでも誘導してください


●使い方
1, 左側のコンボボックスから変換形式を選んでください
2, 真ん中のコンボボックスから既存ファイルを上書きするかどうかを選んでください
3, 下のリストビューに変換したいファイルもしくはフォルダをドロップしてください
4, 変換ボタンを押すとダイアログが出て変換されます（中止する場合は中止をクリックしてください）
5, 入力ファイルのあるフォルダと同じ場所にDDSに変換されたファイルが出力されます

・ドロップされたファイルは、その時にコンボボックスで指定されている変換形式が適用されます
・リストを右クリックするとリストから削除したりすべて選択したり選択したファイルの変換形式を変更出来たりします
・出力ファイル名はリストの出力ファイル名を二回クリックすると変更できます
